# R.I.D. Design System — Tactical Intelligence HUD

> Generated by Stitch (project `7482836304465455378`). This is the reference spec for the Next.js UI rebuild.

## Creative North Star: "The Synthetic Observer"

Inspired by aerospace HUDs and deep-space telemetry consoles. Mission-critical, high-contrast, hyper-precise. **Functional Brutalism** — every element looks engineered for a data-processing purpose.

## Typography

| Role | Font | Weight | Rules |
|------|------|--------|-------|
| Display/Headlines | Space Grotesk / JetBrains Mono | 400-700 | UPPERCASE, `0.05em` letter-spacing |
| Body/Titles | Inter | 400-500 | Default readable |
| Labels | Space Grotesk | 400 | Accompany with index numbers `[01]` |
| Data readouts | JetBrains Mono | 400 | Monospace prevents number jumping |

## Component Rules

### Corner Brackets (not borders)
- Use `primary_fixed` (`#7DF4FF`) for 4px "L" shaped wireframe brackets at container corners
- Never use 1px solid borders to define regions

### Glow Outlines
- Active: `1px border #00F0FF` + `box-shadow: 0 0 8px 0 rgba(0, 240, 255, 0.3)`
- No standard CSS shadows — use glow or brightness

### Buttons
- **Primary:** Solid `#00F0FF`, black text, 0px border-radius
- **Secondary:** Ghost, 1px `#849495` border, cyan text, hover fills 10% glow
- **States:** 0.1s glitch/flicker transition on active

### Badges
- Rectangular, no rounded corners, `label-sm` typography
- JournalClub: green | TLDR: amber | Method: purple | Instagram: indigo | Tool: green

### Cards
- No divider lines within cards
- Corner bracket wireframe accents
- 0px border-radius everywhere

## Visual Effects

- **Scan-line overlay:** 20% opacity horizontal 2px lines (CRT simulation)
- **Glow pulse:** Opacity fade 1→0.6→1 for pulsing neon elements
- **Glitch:** Stochastic clip-path + translate for RGB shift
- **Float:** Vertical bob on skill badges
- **Grid perspective:** `perspective(600px) rotateX(55deg)` with cyan line gradient

## Do's and Don'ts

### Do
- 0px border-radius everywhere (sharp = precision)
- Intentional asymmetry (sidebar shorter than main)
- "Data scaffolding" — decorative `+` symbols, loading bars that never reach 100%
- Dense information, high-bandwidth feel

### Don't
- Standard shadows (use glow instead)
- Icons without labels
- Soft transitions (use step eases or instant)
- Colors outside the palette
- Rounded corners (destroys OLED feel)
